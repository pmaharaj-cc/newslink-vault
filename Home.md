# Trinidad News Knowledge Base

Daily articles from Trinidad Express, processed into linked Obsidian notes.

## Structure
- `Articles/` — one note per article
- `Daily/` — daily index
- `Entities/People/` — named individuals
- `Entities/Orgs/` — organizations
- `Entities/Places/` — locations
- `Entities/Authors/` — journalists
- `Topics/` — topic nodes
